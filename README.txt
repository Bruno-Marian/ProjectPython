- FEITO ISSO APROVEITE O SISTEMA.

sobre o sistema:
	ao entrar no sistema você possui o botão cadastro de cliente, lá você pode "Salvar" novos clientes, "Pesquisar" por clientes cadastrados e verificar o "Clima" dos clientes cadastrados. Para verificar o clima você deve clicar em pesquisar, irá abrir uma tela com uma lista vazia, basta clicar em pesquisar, ira mostrar todos clientes cadastrados, você pode dar um double_click para pegar o cliente. Após pegar o cliente você pode clicar em "Clima" irá trazer algumas informações do clima. Caso esteja errado alguma informação do cadastro ele vai trazer o clima de São Paulo como padrão.
	Caso você edite e salve ele irá editar a informação do usuario e clicar um update.txt de log com os dados alterados e o horário que foi alterado.
	Se clicar em relatório ele ira trazer todos os clientes cadastrados por ordem de cadastro.
